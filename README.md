# 🔗 URL Shortener Project

This project is developed as part of the Industrial Internship program conducted by Upskill Campus in collaboration with UniConverge Technologies Pvt Ltd.

## 📌 Project Overview
The URL Shortener is a web-based application built using Python and Flask. It converts long URLs into short and manageable links, making them easier to share and use.

---

## 🚀 Features
- Convert long URLs into short links  
- Unique short code generation  
- Fast redirection to original URL  
- Simple and user-friendly interface  
- Lightweight and efficient system  

---

## 🛠️ Technologies Used
- Python  
- Flask  
- HTML (for interface)  

---

## ⚙️ How It Works
1. User enters a long URL  
2. System generates a unique short code  
3. Mapping is stored in the database  
4. Short URL is generated  
5. On clicking the short URL → user is redirected to the original URL
